● Add SSH rule for VPN network:

  # Allow SSH from entire VPN subnet
  sudo ufw allow from 10.9.0.0/24 to any port 22

  # Or specifically from Windows VPN IP
  sudo ufw allow from 10.9.0.1 to any port 22

  Check current rules:
  sudo ufw status numbered

  If SSH rule exists but still not working:

  1. Check SSH is listening on VPN interface:
  sudo netstat -tlnp | grep :22
  2. Check SSH config allows VPN connections:
  sudo nano /etc/ssh/sshd_config
  2. Make sure it has:
  ListenAddress 0.0.0.0
  # or
  ListenAddress 10.9.0.2
  3. Restart SSH service:
  sudo systemctl restart ssh
